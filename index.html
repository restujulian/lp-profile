Simple

Solusi yang diberikan oleh Wabi Teknologi harus menyederhanakan proses, bukan membuat proses menjadi lebih rumit. Kesederhanaan juga menjadi inti dari berbagai produk yang dihasilkan. Pelatihan bisa saja merupakan pelatihan dengan materi yang sulit, tetapi materi hharus dirancang dan diimplementasikan sedemikian rupa sehingga terasa lebih sederhana, membumi, dan mudah dipahami.

Interconnected

Solusi yang diberikan oleh Wabi Teknologi merupakan solusi yang mengkoneksikan berbagai konstruksi maupun subsistem. Selain itu, solusi dari Wabi Teknologi tidak dimaksudkan untuk membuat klien berada pada suatu situasi yang terasing. Interoperabilitas merupakan kunci utama.

Valuable

Apapun solusi yang dihasilkan, keluaran dari solusi tersebut harus merupakan keluaran yang berharga, tepat sasaran dalam menyelesaikan masalah klien.

INFORMASI

PT Wabi Teknologi Indonesia atau Wabi Teknologi bermula dari perguruan tinggi bidang informatika dan komputer di Yogyakarta yaitu STMIK AKAKOM. Yayasan Pendidikan Widya Bakti (YPWB) selaku badan penyelenggara pendidikan untuk institusi STMIK AKAKOM mengembangkan lini bisnis di luar pendidikan tinggi untuk mengantisipasi berbagai kebutuhan di bidang informatika dan komputer yang berkembang di masyarakat.

Unit pengembangan usaha ini akan menjadi suatu unit dengan hubungan mutual relationship dengan STMIK AKAKOM. Sumber daya yang ada di STMIK AKAKOM bisa digunakan untuk kepentingan pengembangan usaha mengingat Ketersediaan SDM terkait sistem informasi di STMIK AKAKOM yang cukup melimpah dengan skillset yang baik.

Wabi Teknologi mulai dibentuk tahun 2018 dan memperoleh legalitas untuk beroperasi pada bulan Agustus 2018. Sebagai suatu badan hukum berbentuk PT (perseroan terbatas), Wabi Teknologi mempunyai AD/ART yang telah disahkan melalui akta notaris:

Akta No 3 Tanggal 13 Agustus 2018, Notaris Muchammad Agus Hanafi, S.H.

Pelayanan

Saat ini Wabi Teknologi bergerak di beberapa bidang dalam teknologi informasi. Ruang lingkup layanan ada 2, yaitu Engineering (memproduksi software) dan Education (melaksanakan berbagai pelatihan). Pelatihan bisa dari in-house (dibuat sendiri oleh Wabi Teknologi), maupun bekerjasama dengan vendor).

Engineering

Divisi ini memberikan Layanan pembuatan software :

  • Pembuatan custom software untuk suatu organisasi
  • Pembuatan produk software _mass software
  • Pembuatan produk software berupa software untuk masyarakat (Mobile App) dengan penghasilan dari monetisasi serta komisi dari transaksi masyarakat pemakai
  • Layanan software non sistem informasi: AI, Big Data Analytics

Education

Layanan yang diberikan oleh divisi ini meliputi layanan pelatihan berikut ini :

  • Pelatihan untuk instansi
  • Pelatihan untuk umum
  • Pelatihan custom untuk organisasi
  • Pelatihan privat untuk umum

Jasa (Engineering)

Saat ini ruang lingkup software yang dikembangkan meliputi berbagai platform dan tidak terbatas pada bidang tertentu. Secara umum, sumber daya Wabi Teknologi diarahkan untuk membangun software :

Aplikasi Web

Aplikasi Web, pada sisi client (front end) maupun server (back end).

Aplikasi mobile

Aplikasi mobile, baik pada sisi client maupun mobile backend. Wabi Teknologi menggunakan peranti pengembangan Kotlin / Java untuk Android.

Aplikasi komputasi ilmiah

Aplikasi komputasi ilmiah (scientific computing). Wabi Teknologi menggunakan tools Python (Anaconda dengan pustaka SciPy, NumPy, dan lain-lain) dan Julia untuk keperluan ini.

Data Analytics

Wabi menggunakan SQL, NOSQL, dan NewSQL untuk mengolah data (besar) serta menghasilkan pola-pola maupun informasi yang bisa digunakan untuk pengambilan keputusan. Di sisi peranti pengembangan, Wabi Teknologi menggunakan R, Python, serta berbagai tools di platform Java.

Artifical Intelligence

Artifical Intelligence menggunakan TensorFlow dan berbagai pustaka peranti pengembangan lainnya.

Sistem Terdistribusi

Aplikasi sistem terdistribusi menggunakan Go / Rust.

Internet of Thing

Aplikasi IoT (Internet of Thing).

Sistem Terdesentralisasi / Blockchain

Aplikasi sistem terdesentralisasi / blockchain (private maupun DApp untuk public blockchain). DApp meliputi public blockchain maupun private blockchain.
